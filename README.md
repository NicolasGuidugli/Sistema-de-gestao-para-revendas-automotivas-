# Projeto Gestão Revenda Automotiva

Sistema SaaS de gestão para revendas automotivas
Esse sistem tem por finalidade gerenciar o estoque, o financeiro, documentar as especificações dos veiculos na revenda,
como por exemplo os adicionais que o veiculo possui, e também documentar as manutenções feitas como oque foi reparado,
e também registrar da onde veio quem forneceu e para quem foi vendido.

##  Tecnologias planejadas
- **Banco de dados:** PostgreSQL (NeonDB)
- **Backend:** Java + Spring Boot + JWT + BCrypt + JPA
- **Frontend:** React + JavaScript + HTML + CSS

##  Estrutura
- `database/historico/Final-Database` — Evolução do banco de dados (Finalizado)
- `backend/` — API REST (em desenvolvimento)
- `frontend/` — Interface web (em desenvolvimento)
- `docs/` — DER e documentação
