# Projeto Gestão Revenda Automotiva

Sistema SaaS de gestão para revendas automotivas
Esse sistem tem por finalidade gerenciar o estoque, fazer relátorios financeiro, controlar as compras e vendas, entradas e saídas de veículos,
documentar as especificações dos veiculos na revenda, como por exemplo os adicionais que o veiculo possui, 
e também documentar as manutenções feitas como oque foi reparado, registrando as receitas e as despesas,
também será possível registrar da onde veio quem forneceu e para quem foi vendido, controle de clientes e controle de fornecedores.
Haverá login seguro com JWT, senhas criptografadas e controle de acesso com permissões.

# Objetivo

Fornecer uma plataforma moderna, segura e escalável para revendas automotivas, 
centralizando a gestão operacional, financeira e documental em um único sistema.

##  Tecnologias Utilizadas
- **Banco de dados:** PostgreSQL (NeonDB)
- **Backend:** Java + Spring Boot + JWT + BCrypt + JPA
- **Frontend:** React + JavaScript + HTML + CSS

##  Estrutura
